# Windows 11 Repair Tool

Windows 11 repair PowerShell tool that runs DISM and SFC, creates logs, supports optional ISO/WIM/ESD source repair, and helps troubleshoot corrupted system files.

## Features
- Automatic administrator elevation
- Quick repair mode
- Full repair mode
- Optional source-based DISM repair
- Log file output
- Reboot prompt after repair
